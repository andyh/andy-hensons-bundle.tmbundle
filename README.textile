h1. Installation

To install with Git:


@mkdir -p ~/Library/Application\ Support/TextMate/Bundles@
@cd ~/Library/Application\ Support/TextMate/Bundles@
@git clone git://github.com/andyh/andy-hensons-bundle.tmbundle.git "Andy Henson's Bundle.tmbundle"@
@osascript -e 'tell app "TextMate" to reload bundles'@

Or use Yehuda's TextMate commandline package manager ("http://github.com/wycats/textmate/tree":http://github.com/wycats/textmate/tree)
